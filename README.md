## What Liquid Rhythm Does

Liquid Rhythm is designed for creators who need a quicker path from rhythmic idea to usable production material. Instead of splitting rhythm design across multiple utilities, it centers timing structure, pattern building, and direct editing inside a single desktop workflow.

The application is intended for producers and audio-focused setups where DAW connectivity, MIDI export, and visual control are important. With adaptive grid behavior, waveform-driven rhythm extraction, and collaboration support, it works for both independent sketching and shared session work.

## Using the App

The usual workflow is centered on creating rhythms, shaping them interactively, and moving the result into a production chain.

1. Import or create a rhythm source.
2. Use the adaptive grid to shape meter changes and pattern timing.
3. Edit the pattern with gestures for faster refinement.
4. Extract rhythm from waveform material when working from audio.
5. Send the finished output through MIDI to your DAW or other music software.
6. Save your session to preserve backups and recovery data.

Example workflow:

- Start a new rhythm session
- Choose a time structure
- Add or modify gestures
- Sync with collaborators if needed
- Export MIDI or continue in the DAW
